# Removed
Contains ports that have been deleted from the Ravenports tree.

### Setup

This script is expected to be run on a system that have both the
Ravenports/Ravensource and the Ravenports/ravensource repositories.

1. Copy config.yaml.sample to config.yaml
1. Define the path to the Ravenports repository in config.yaml file
1. Define the path to the ravensource repository in the config.yaml file

#### Install packages

Ensure specific python modules are installed:

- python-PyYaml

Ensure this software is available too:

- git
- rsync

### Latest list of deleted ports

See the alphabetical [list of deleted ports](history.md).  It shows the last commit
hash and date the port was present in the Ravenports tree.
